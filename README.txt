Data logging using raspberry pi and bit wizard UI

Components 
- Upload data from CSPro (PDA) or ODK (Android)
- Enter data using keyboard entry
- Log sensor values based on entered/uploaded data
- Upload data to MySQL sever via, JSON, Requests and Pyramid

To come
- Power management
- More on sensors.

This is based on the bit wizard UI platform, managing menus text display and I2C protocol.

Programming is a mix of Python, Shell script and Bash
